# Data licensing and attribution

Project XC separates code, curated metadata, and imported source-derived metadata.

## Code

Repository scripts, website code, and workflow glue are released under the MIT license in `LICENSE`.

## Curated Project XC metadata

Project-authored curation in `data/functionals.seed.json`, `data/program_aliases.json`, and documentation is intended for open scholarly reuse under CC BY 4.0, with attribution to Project XC and the original scientific references.

## Imported Libxc snapshot

`data/libxc_snapshot.json` is generated from the public Libxc functionals documentation at:

https://libxc.gitlab.io/functionals/

Libxc is described by its website as released under the Mozilla Public License 2.0. Project XC imports names, Libxc IDs, descriptions, and reference links for discovery/provenance. Users redistributing or modifying this imported layer should preserve Libxc attribution and respect Libxc/MPL-2.0 terms.

Primary Libxc citation/landing page:

https://libxc.gitlab.io/

## Scientific references

Every functional record should cite the original scientific paper(s). Project XC does not claim ownership of citations, DOI metadata, or paper content.
